# Github Terminal Command Guide

## ⭐️ Commit 하기
#### 1. 모든 수정 파일 대기열에 올리기   
    git add *


#### 2. Commit 메시지 작성하기
    git commit -m "Your message"


#### 3. Push 하기
    git push origin {branch name (Ex: main)}


## Fetch 받기
    git pull origin {branch name (Ex: main)}


## Update 확인하기
    git status
